# Squee & Spoon — Pan-Asian Restaurant Website

A minimal, high-end Next.js website for **Squee & Spoon**, a Pan-Asian restaurant in Seawoods, Navi Mumbai.

## Tech Stack
- Next.js 16 (App Router)
- Tailwind CSS 4
- TypeScript

## Color Palette
| Name | Hex |
|------|-----|
| Midnight Navy | `#03202B` |
| Champagne Gold | `#CEAB79` |
| Ivory White | `#F4EFFA` |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Sections
- **Hero** — Bold minimal typography + logo
- **Menu Highlights** — Ramen, Bao, Sushi
- **About** — Cozy Pan-Asian cafe with indoor/outdoor seating
- **Contact** — Address & phone
- **Fixed CTA** — "Enquire on WhatsApp" button
